# Session Clean — Очистка транскрипта сессии

## Рабочий цикл

1. Получить **манифест** от пользователя (первое сообщение)
2. Уточнить номер сессии и файл транскрипта (переспросить если неясно)
3. Прочитать словарь имён: `Кампейны/Азланти/Glossary/Transcription_Dictionary.md`
4. Прочитать правила очистки: `Кампейны/Азланти/Assistant Guides/11__Очистка_транскрипции.md`
5. Получить чанк (между `=== CHUNK START ===` / `=== CHUNK END ===`)
6. Очистить по правилам
7. Append результат в файл транскрипта текущей сессии
8. Подтвердить: «Чанк записан, жду следующий»

## Манифест сессии

В начале работы пользователь присылает блок метаданных вида:

```
Session folder: C:\DND\Азланти\Cессии\сессия-0
Source file: merged.txt
Speakers (detected from merged.txt):
- Сеня (Микаэль/Анканто)
- Вова
- ...
Notes:
- Chunks overlap on purpose: repeated paragraphs help preserve context across boundaries.
- When summarizing chunk-by-chunk, deduplicate overlaps while merging the summaries.
```

Из манифеста берём:
- **Speakers** — маппинг «кто говорит → кто персонаж». Сверять со словарём.
- **Session folder** — только для определения номера сессии (сессия-0 → Session0). Переспросить у пользователя если неясно куда писать.
- **Notes** — чанки перекрываются: при append дедуплицировать повторы на стыках.

## Где что лежит

| Что | Путь (Obsidian) |
|-----|-----------------|
| Словарь имён | `Glossary/Transcription_Dictionary.md` |
| Правила очистки | `Assistant Guides/11__Очистка_транскрипции.md` |
| Папка сессий | `Sessions/Session0/`, `Sessions/Session1/` и т.д. |
| Файл транскрипта | `Sessions/SessionN/ПСN_transcript.md` или `Sessions/SessionN/transcript.md` |

Все пути — относительно `Кампейны/Азланти/` в Obsidian.

## Краткие правила (полные — в гайде)

### Исправляем
- Имена → по словарю
- Споры о механике → только итог
- Ошибки распознавания → по контексту

### Убираем
- Мета вне игры (споры +7 или +3, билды, ссылки на книги)
- Перерывы, болтовня ни о чём
- Рефлексия после сессии
- Техпроблемы (звук, микрофон)

### Оставляем (максимально подробно)
- Весь отыгрыш, описания ГМ, диалоги
- Тактику, боёвки, броски, решения
- Всё что in-character или in-game

### Формат вывода
- `**Имя (роль):**` — кто говорит
- Между сценами — `---`
