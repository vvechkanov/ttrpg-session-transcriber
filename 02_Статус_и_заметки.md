# discord-session-transcriber — Статус и заметки

## Что уже стабильно
- Автономный инсталлятор (venv + PyTorch + WhisperX + ffmpeg — всё внутри проекта).
- Запуск через `run.bat` — GUI на tkinter.
- Разбор отдельных дорожек WhisperX (модель `large-v3`, GPU/CPU).
- Склейка в единый `merged.txt` с поддержкой `speaker_map.json`.
- Нарезка `merged.txt` на чанки с overlap.

## Что в процессе
- Тонкая фильтрация мусора/галлюцинаций распознавания (типа «Редактор субтитров…»).
- Финальный этап «конспект → журнал» по вкусу/ритму (стиль ещё будет дорабатываться).

## Текущая цель
Держать единый повторяемый цикл: аудио → конспект → журнал.
Проект полностью автономен — не зависит от внешних папок.
