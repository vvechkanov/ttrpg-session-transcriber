# ttrpg-session-transcriber — Статус и заметки

**Обновлено:** 2026-06-12 (архитектурное ревью).

## Что уже стабильно

- Шестислойная архитектура (`domain / sources / mergers / renderers / core / ui`) — внедрена, см. `ARCHITECTURE.md`.
- GUI на PySide6/QML (`python -m ui`): таймлайн, волны, фазы пайплайна, настройки.
- ASR-бэкенды: faster-whisper (default), GigaAM-v3 (sherpa-onnx), WhisperX (legacy).
- Несколько Craig-архивов в одной сессии (per-segment ASR + waveforms).
- Редактор speaker_map (игрок → персонажи, ГМ, слушатель) + CastStrip.
- Склейка в `merged.txt` с интерливингом FVTT-чата; нарезка на LLM-чанки с overlap.
- CLI с паритетом GUI (`ui/cli.py`).
- ~440 тестов, CI-матрица (ubuntu/windows × py3.11/3.12), 3-уровневая e2e-стратегия.

## Что в процессе / на очереди
